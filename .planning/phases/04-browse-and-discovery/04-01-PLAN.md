**Also update `lib/data/index.ts`** — add a `getDistinctRegions` helper:
```typescript
export async function getDistinctRegions(): Promise<string[]> {
  const results = await prisma.property.findMany({
    select: { region: true },
    distinct: ['region'],
    orderBy: { region: 'asc' },
  });
  return results.map(r => r.region);
}
```

This helper is needed by the filter bar in Plan 04-03.
