  <action>
REPLACE the placeholder `components/ai-chat/ai-chat-client.tsx` (created in Plan 01) with the full client component.

**Implementation:**

1. `"use client"` directive at top.

2. Imports:
   - `useState`, `useRef`, `useEffect`, `useMemo` from "react"
   - `Sparkles`, `Send` from "lucide-react"
   - `Button` from `@/components/ui/button`
   - `Input` from `@/components/ui/input`
   - `TypingIndicator` from `@/components/messaging/typing-indicator`
   - `AIMessage` from `./ai-message`
   - `UserMessage` from `./user-message`
   - Type import: `import type { getAllProperties } from "@/lib/data"`

3. Types:
   ```typescript
   type Property = Awaited<ReturnType<typeof getAllProperties>>[number]
   type UserMsg = { role: "user"; content: string }
   type AIMsg = { role: "assistant"; content: string; recommendedSlugs: string[] }
   type ChatMessage = UserMsg | AIMsg
   ```

4. Props: `{ properties: Property[] }`

5. State:
   - `messages: ChatMessage[]` — initialized empty
   - `inputValue: string` — empty
   - `isLoading: boolean` — false
   - `error: string | null` — null

6. Refs:
   - `scrollRef` for the message container div

7. `useMemo` for property slug lookup map:
   ```typescript
   const propertyMap = useMemo(
     () => new Map(properties.map((p) => [p.slug, p])),
     [properties]
   )
   ```

8. `useEffect` for scroll-to-bottom on `[messages, isLoading]` — same pattern as ConversationView:
   ```typescript
   if (scrollRef.current) {
     scrollRef.current.scrollTop = scrollRef.current.scrollHeight
   }
   ```

9. `handleSend(content: string)` async function:
   - Guard: if `!content.trim() || isLoading` return
   - Create user message: `{ role: "user", content: content.trim() }`
   - Build updated messages array (existing + new user message)
   - Clear input, clear error, set messages to updated array, set isLoading true
   - `fetch("/api/ai-chat", { method: "POST", headers: { "Content-Type": "application/json" }, body: JSON.stringify({ messages: updatedMessages.map(m => ({ role: m.role, content: m.content })) }) })`
   - **IMPORTANT:** When sending messages to the API, strip `recommendedSlugs` — only send `{ role, content }` for each message. Prior recommendations should NOT be re-sent as context.
   - On success: parse response, create AI message `{ role: "assistant", content: data.text, recommendedSlugs: data.recommendedSlugs ?? [] }`, append to messages
   - On error: set error message "Failed to get a response. Please try again."
   - Finally: set isLoading false

10. `handleKeyDown` — Enter (not Shift+Enter) triggers handleSend

11. Render structure (full-height flex column):
    ```
    <div className="flex flex-col h-full">
      {/* Header */}
      <div className="border-b px-4 py-3 shrink-0">
        <div className="flex items-center gap-2">
          <Sparkles className="h-4 w-4 text-primary" />
          <h1 className="font-semibold text-sm">Anvara AI</h1>
        </div>
        <p className="text-xs text-muted-foreground mt-0.5">
          Describe what you're looking for and I'll recommend sponsorship opportunities
        </p>
      </div>

      {/* Messages area */}
      <div ref={scrollRef} className="flex-1 overflow-y-auto p-4 space-y-3">
        {messages.length === 0 ? (
          /* Empty state */
          <div className="flex-1 flex flex-col items-center justify-center h-full gap-3 text-center">
            <Sparkles className="h-8 w-8 text-primary" />
            <h2 className="text-lg font-semibold">Anvara AI</h2>
            <p className="text-sm text-muted-foreground max-w-xs">
              Describe what you're looking for and I'll find relevant sponsorship opportunities for you.
            </p>
          </div>
        ) : (
          messages.map((msg, i) =>
            msg.role === "user" ? (
              <UserMessage key={i} content={msg.content} />
            ) : (
              <AIMessage
                key={i}
                content={msg.content}
                recommendedProperties={
                  msg.recommendedSlugs
                    .map((slug) => propertyMap.get(slug))
                    .filter((p): p is Property => p !== undefined)
                }
              />
            )
          )
        )}
        {isLoading && <TypingIndicator />}
      </div>

      {/* Error */}
      {error && (
        <div className="px-4 py-2 border-t shrink-0">
          <p className="text-xs text-red-600 dark:text-red-400">{error}</p>
        </div>
      )}

      {/* Input area */}
      <div className="border-t p-4 shrink-0">
        <div className="flex items-center gap-2">
          <Input
            value={inputValue}
            onChange={(e) => setInputValue(e.target.value)}
            onKeyDown={handleKeyDown}
            placeholder="Ask about sponsorship opportunities..."
            disabled={isLoading}
            className="flex-1"
          />
          <Button
            type="button"
            size="icon"
            onClick={() => handleSend(inputValue)}
            disabled={!inputValue.trim() || isLoading}
            aria-label="Send message"
          >
            <Send className="h-4 w-4" />
          </Button>
        </div>
      </div>
    </div>
    ```

**DO NOT:** Use `useChat` from `@ai-sdk/react` (not installed). Do NOT persist messages to database. Do NOT copy PropertyCard or TypingIndicator — import from existing locations. Do NOT show raw JSON to user — always display only `parsed.text`.
  </action>
