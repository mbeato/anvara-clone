  <name>Task 1: Create AI chat API route and typing indicator</name>
  <files>
    app/api/chat/route.ts
    components/messaging/typing-indicator.tsx

  <action>
1. Create `app/api/chat/route.ts` — POST handler for AI response generation:
   - `export const maxDuration = 60` — o3-mini needs extended timeout
   - Import `generateText` from "ai", `openai` from "@ai-sdk/openai", prisma from "@/lib/prisma"
   - Accept POST body: `{ threadId: string }`
   - Fetch thread with messages + property + packages via Prisma findUnique:
     ```
     prisma.thread.findUnique({
       where: { id: threadId },
       include: {
         messages: { orderBy: { createdAt: "asc" } },
         property: { include: { packages: true } }
       }
     })
     ```
   - Return 404 if thread not found
   - Build messages array mapping thread messages to `{ role, content }`:
     - sender "advertiser" → role "user"
     - sender "property" → role "assistant"
   - Build system prompt via `buildSystemPrompt(thread.property)` helper (define inline in same file):
     - Include property name, category, location
     - Include audience reach, age range, gender
     - Include first 500 chars of description
     - List all packages with name, price, first 2 inclusions
     - Tone: "You are a sponsorship sales representative for {name}. Respond professionally and knowledgeably. Reference actual property details and tier names. Keep responses concise (2-4 short paragraphs)."
   - Call `generateText({ model: openai("o3-mini"), system: systemPrompt, messages: aiMessages })`
   - Save AI reply to DB: `prisma.message.create({ data: { threadId, content: text, sender: "property", isAI: true } })`
   - Return `Response.json({ message: aiMessage })`
   - Wrap in try/catch — on error return `Response.json({ error: "Failed to generate response" }, { status: 500 })`

2. Create `components/messaging/typing-indicator.tsx`:
   - Three dots with staggered bounce animation
   - Wrapped in a left-aligned bubble container (same positioning as property messages)
   - Use Tailwind animate-bounce with `[animation-delay:-0.3s]`, `[animation-delay:-0.15s]`, and no delay
   - Dots: `h-2 w-2 rounded-full bg-muted-foreground/60`
   - Container: gray bubble matching property message style (`bg-muted rounded-2xl rounded-bl-sm px-4 py-3`)
  </action>

    - `components/messaging/typing-indicator.tsx` exists
    - Check that OPENAI_API_KEY is in .env.local (log reminder if not — the user_setup handles this)
  </verify>
